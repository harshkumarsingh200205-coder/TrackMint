# TrackMint

TrackMint is a Java-based expense tracker that helps users record daily spending, store data persistently, and analyze expenses.

## Features

- User registration and login
- Add, edit, delete expenses
- Category-wise tracking
- Budget setting
- Expense analytics
- Report export

## Tech Stack

- Java
- JDBC
- SQLite
